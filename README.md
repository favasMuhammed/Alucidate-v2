<details open>
  <summary><b>📖 Overview</b></summary>
  <br/>
  <blockquote>
    <b>AIucidate</b> is an intelligent, offline-capable academic study assistant designed to transform static PDF textbook chapters into interactive learning modules. The application provides students with AI-generated summaries, keyword definitions, and hierarchical mind maps for every chapter.
  </blockquote>
  
  The application features role-based workflows:
  - 👨‍🏫 **Administrators**: Upload and manage subject syllabuses by uploading PDF chapters. The app automatically analyzes the PDFs and generates interactive, structured study content.
  - 🎓 **Students**: Access their class materials, explore interactive mind maps, and chat with a context-aware AI tutor that answers questions accurately based *only* on the provided textbook material.
</details>

<br/>

  <tr>
    <td width="50%">
      <h3 align="center">📊 Hierarchical Mind Maps</h3>

    </td>
    <td width="50%">
      <h3 align="center">💾 Offline-First IndexedDB</h3>

### Role-based Authentication Flow
Effortless, dynamic, and localized signup and login flows utilizing in-browser storage techniques to switch between the robust **Admin** and **Student** dashboards in milliseconds.

## 🚀 Instant Local Setup

<details>
<summary><b>Click to expand setup instructions</b></summary>
<br/>

**1. Clone the repository**:
```bash
git clone https://github.com/favasMuhammed/Alucidate-v2.git
cd Alucidate-v2
```

**2. Hydrate dependencies**:
```bash
npm install
```

**3. Configure Secure Environment**:
Create a `.env.local` file in the root directory and inject your securely obtained API key.
```env
GEMINI_API_KEY=your_api_key_here
```

**4. Ignite the Development Server**:
```bash
npm run dev
```
> The application will instantly hot-reload at `http://localhost:3000/`.

</details>
